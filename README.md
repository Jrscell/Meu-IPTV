# Meu IPTV - Site de Streaming

Este é um projeto de **site IPTV** desenvolvido como parte do meu aprendizado em desenvolvimento web. O site permite que os usuários acessem e assistam a conteúdos de vídeo ao vivo por meio de uma interface simples e funcional.

**Funcionalidades:**
- **Exibição de conteúdo ao vivo**: Carregamento de playlists IPTV para exibição de vídeos ao vivo.
- **Interface simples e intuitiva**: Focada na experiência do usuário.
- **Responsividade**: O layout se adapta automaticamente a diferentes dispositivos, como celulares, tablets e desktops.
- **Controle de vídeo**: Os usuários podem controlar a reprodução (play/pause) e ajustar o volume.

**Tecnologias Utilizadas:**
- **HTML**: Estrutura do site e conteúdo estático.
- **CSS**: Estilização e responsividade.
- **JavaScript**: Funcionalidade dinâmica, incluindo o controle de playlists e vídeos.
- **API de IPTV**: Para carregar canais de TV ao vivo.
